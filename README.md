# ✦ Velvet — Private. Elegant. Yours.

> A luxury, end-to-end encrypted messaging platform. No phone number. No email. Just your unique Velvet code.

**Built by Velvet Pages · Founder: Mayukh Sarkar**

---

## 🌟 What is Velvet?

Velvet is a private messaging web app that gives every user a **unique 5–7 digit code** as their identity. No personal information required. No sign-up friction. Just connect, chat, and express yourself — beautifully.

---

## ✦ Features

| Feature | Description |
|---|---|
| 🔒 **End-to-End Encrypted** | AES-256 E2EE on all messages, media, and groups |
| ⬡ **No Phone / Email** | Only a unique Velvet code is needed |
| 🐾 **Pet Companion System** | Choose your spirit animal from 25+ pets |
| 🎨 **Custom Backgrounds** | Presets, solid color, gradient, or your own image |
| ✦ **Animated Typos** | Unique library of typographic symbols |
| 📝 **Multiple Fonts** | Default, Serif, Elegant, Mono, Italic, Light |
| 🎮 **In-Chat Games** | Tic-Tac-Toe, Rock-Paper-Scissors, Truth or Dare |
| 👥 **Group Chats** | Create groups, add by Velvet code |
| 📎 **All File Formats** | Images, videos, documents — all supported |
| 🙈 **Name Hiding** | Hide your name, show only your Velvet code |
| 🖼 **Profile Pictures** | Upload custom profile photos |
| 🛡️ **Admin Panel** | Secure admin dashboard with special access code |

## 🚀 Deploy on GitHub Pages

1. Fork or upload this repo
2. Go to **Settings → Pages**
3. Set source to `main` branch, `/ (root)`
4. Your app is live at:
   - App: `https://yourusername.github.io/velvet/`
   - Admin: `https://yourusername.github.io/velvet/admin.html`

---

## 🔑 Admin Panel Access

The admin panel (`admin.html`) is protected by a special code.

Default code: **`VELVET2024`**

> ⚠️ Change this in `admin.html` before deploying publicly. Search for `ADMIN_CODE` in the script.

**Admin panel features:**
- Dashboard with live stats & charts
- User registry & activity feed
- User reviews management
- Feature & "Why Velvet" pages
- Platform toggle settings
- Help & support links
